-网络流-最大流-二分图- [洛谷 PP3254][网络流24题]圆桌问题
发表于 2019-05-24| 更新于 2019-08-08

题目描述

Link

Solution

建立二分图和超级源汇，将每个源点连向每个单位的编号，容量为这个单位的人数，将每个圆桌的编号连向超级汇点，容量为这个圆桌所能容纳的人的数量，再在每个单位和每个圆桌分别相连，容量为 $1$。

最后跑最大流就可以了。
