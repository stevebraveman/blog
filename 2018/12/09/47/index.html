发表于 2018-12-09| 更新于 2019-08-08

题目描述

n个学生去p个课堂，每一个学生都有自己的课堂，并且每个学生只能去一个课堂，题目要求能够安排每一个课堂都有人吗？

输入格式

第一行是测试数据的个数，

每组测试数据的开始分别是p和n，

接着p行，每行的开始是这个课堂的学生人数m，接着m个数代表该课堂的学生编号

输出格式

如果该组数据能够这样安排就输出YES，否则输出NO。

输入样例

2
3 3
3 1 2 3
2 1 2
1 1
3 3
2 1 3
2 1 3
1 1

输出样例

YES
NO

说明

对于100%的数据，$n\le 100,m\le 20000$
